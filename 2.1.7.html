Describe which are the appropriate filetypes for websites


The appropriate filetypes to use for a website are VI — Audio/Video Interleaved.as some website have video and audio content on the website.


CSS — Cascading Stylesheet is an appropriate file type as this file type help you make your website look pretty, this is where you will sort the colour scheme for the whole website


DOC — Microsoft Word Document is an appropriate file type as so website has attachments so that people can download documents


EXE — Executable file is appropriate as is used to perform various functions or operations on a computer. Unlike a data file, an executable file cannot be read because it's compiled. On an IBM compatible computer.


Graphics Interchange Format is an appropriate file as it is bitmap image format which means all images are mainly GIF images used on website


Hypertext Mark-up Language files is an appropriate file as it is the most basic building blocks to create a website


Musical Instrument Digital Interface. Is appropriate as some website purposes have to music based so have music instrument digital interface help record music to put on the website
